# Exam Quiz App

React + Vite 기반 객관식 학습 앱입니다.

## 주요 기능

- `.xlsx` 문제 파일 업로드
- 랜덤 사이클 출제 (사이클 문제 수 지정 가능)
- 우선순위 출제:
  - 미풀이 문제 최우선
  - 오답률 높은 분야 문제 차우선
- 정답/오답 하이라이팅
  - 정답: 녹색
  - 오답: 적색

## 엑셀 컬럼 형식

다음 컬럼명을 권장합니다.

- `question` (문제)
- `choice1` ~ `choice5` (보기)
- `answer` (정답 번호: 1부터 시작)
- `category` (분야)

한국어 컬럼도 지원합니다.

- `문제`
- `보기1` ~ `보기5`
- `정답`
- `분야`

## 실행

```bash
npm install
npm run dev
```
